# SecReport
ChatGPT加持的，多人协同信息安全渗透测试报告`编写`/`导出`平台

[官方网站](https://sec-report.com)

* 标准化渗透测试流程
* 多人协同编辑
* 自定义导出模版
* ChatGPT生成漏洞简介及修复方案

## 社区单机版安装

```shell
git clone https://github.com/sec-report/SecReport
cd SecReport
chmod +x run.sh
./run.sh
```

Docker全部运行后访问: [http://127.0.0.1](http://127.0.0.1/)
